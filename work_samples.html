1. Custom VPN on an EC2 Server in AWS

To demonstrate my knowledge of AWS, I created a custom VPN hosted on an EC2 server and published the instructions to do so in my GitHub repository.

The steps included spinning up a Ubuntu instance, allowing SSH connections over port 22 in the instance's Security Group, and creating a pair of security keys for the SSH login.

Then, I configured Security Groups to add an inbound rule to allow traffic from UDP port 1194.

In the SSH terminal, I ran the following commands from the Readme in the openvpn-install repository: 

            sudo curl -O https://raw.githubusercontent.com/angristan/openvpn-install/master/openvpn-install.sh
            sudo chmod +x openvpn-install.sh
            sudo ./openvpn-install.sh

After the installation was complete, an .OVPN file was created in the current EC2 directory. To use the VPN, I imported the .OVPN file into OpenVPN and connected to the server.
